# Enterprise Supply Chain Optimization & Operations Intelligence Dashboard

## Project Overview
This project analyzes enterprise supply chain performance using Excel, Python, SQL, and Power BI.

The dashboard focuses on:
- Revenue and profitability analysis
- Supplier performance evaluation
- Inventory optimization
- Logistics and delivery performance
- Forecasting and strategic recommendations

## Tools Used
- Python
- SQL (MySQL)
- Power BI

## Project Workflow
1. Data cleaning and feature engineering using Python
2. SQL database creation and business query analysis
3. Interactive Power BI dashboard development

## Dashboard Pages
1. Executive Supply Chain Overview
2. Inventory and Supplier Performance
3. Logistics and Delivery Performance
4. Forecasting and Strategic Recommendations

## Key Insights
- Supplier performance impacts delivery efficiency
- Inventory stockout risks identified
- Logistics costs vary significantly by region
- Forecasting supports operational planning
